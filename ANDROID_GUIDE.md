- ✅ 电脑和手机连接到**同一个Wi-Fi网络**
- ✅ 电脑已启动后端和前端服务
- ✅ 防火墙已配置允许访问

你的电脑局域网IP地址：**10.6.6.102**

在安卓手机浏览器中访问：

- **前端界面**: http://10.6.6.102:3000

- **API文档**: http://10.6.6.102:8000/docs

### 详细步骤

#### 1. 确认网络连接
- 确保电脑和手机连接到同一个Wi-Fi
- 检查电脑的Wi-Fi连接状态

#### 2. 启动电脑服务
在电脑上运行以下命令启动服务：

**启动后端：**


打开安卓手机的浏览器（Chrome、Edge等），输入：

## 🌐 方法二：使用内网穿透工具

### 使用ngrok（推荐）

#### 1. 下载并安装ngrok
访问 https://ngrok.com 下载并安装ngrok

#### 2. 启动ngrok
在电脑上运行：
```bash
ngrok http 3000
```

#### 3. 获取公网地址
ngrok会提供一个公网地址，例如：
```
https://abc123.ngrok.io
```

#### 4. 在手机上访问
在安卓手机浏览器中访问ngrok提供的地址

#### 5. 配置后端访问
需要同时为后端创建ngrok隧道：
```bash
ngrok http 8000
```

### 使用其他内网穿透工具

- **花生壳**：https://hsk.oray.com
- **frp**：https://github.com/fatedier/frp
- **localtunnel**：https://localtunnel.github.io

## 📲 方法三：使用静态HTML文件

### 优点
- 无需启动后端服务
- 可以直接在手机上打开
- 体验界面设计

### 缺点
- 无法进行真实的API调用
- 功能受限（仅界面演示）

### 操作步骤

#### 1. 传输HTML文件到手机
使用以下任一方法将 `frontend/index.html` 传输到手机：

**方法A：通过云存储**
1. 将 `frontend/index.html` 上传到云盘（百度网盘、Google Drive等）
2. 在手机上下载并打开

**方法B：通过微信/QQ**
1. 将 `frontend/index.html` 发送到微信/QQ
2. 在手机上保存并打开

**方法C：通过USB连接**
1. 用USB数据线连接手机和电脑
2. 将文件复制到手机存储
3. 在手机文件管理器中打开

#### 2. 在手机浏览器中打开
找到下载的HTML文件，用浏览器打开即可

## 🎯 方法四：使用本地服务器 + 手机热点

### 操作步骤

#### 1. 电脑连接手机热点

#### 3. 启动服务
在电脑上启动后端和前端服务

#### 4. 在手机上访问
使用电脑的IP地址访问（与方法一类似）

## 🔧 方法五：使用VS Code Live Server

### 优点
- 简单快速
- 适合前端界面体验

### 操作步骤

#### 1. 安装Live Server扩展
在VS Code中安装 "Live Server" 扩展

#### 2. 打开HTML文件
在VS Code中打开 `frontend/index.html`

#### 3. 启动Live Server
右键点击HTML文件，选择 "Open with Live Server"

#### 4. 配置网络访问
Live Server默认只监听localhost，需要修改配置：
- 打开VS Code设置
- 搜索 "Live Server"
- 找到 "Live Server > Settings > Host"
- 将 `localhost` 改为 `0.0.0.0`

#### 5. 在手机上访问
使用电脑IP地址访问

## 🚨 常见问题

### 问题1：无法访问局域网地址

**解决方案：**
1. 确认电脑和手机在同一Wi-Fi网络
2. 检查防火墙设置
3. 确认服务正在运行
4. 尝试关闭电脑的防火墙测试

### 问题2：连接超时

**解决方案：**
1. 检查服务是否正常启动
2. 确认端口号正确
3. 尝试重启服务
4. 检查网络连接

### 问题3：CORS错误

**解决方案：**
1. 检查后端CORS配置
2. 确保前端地址在允许列表中
3. 修改 `backend/app/core/config.py` 中的 `cors_origins`

### 问题4：页面显示异常

**解决方案：**
1. 清除浏览器缓存
2. 更新浏览器到最新版本
3. 尝试使用其他浏览器
4. 检查网络连接速度

### 问题5：语音功能不工作

**解决方案：**
1. 确认浏览器支持Web Speech API
2. 检查麦克风权限
3. 确保麦克风正常工作
4. 尝试使用HTTPS连接（某些浏览器要求）
